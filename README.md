# WTWR React

## Description

WTWR React is a weather based clothing recommendation app built with React and Vite. It displays the current weather for a set location and suggests clothing items that match the weather conditions. Users can open a modal to add a new garment and click clothing cards to view them in a preview modal.

## Functionality

- Fetches current weather data from the OpenWeather API
- Parses weather data and categorizes it as hot, warm, or cold
- Displays clothing items filtered by weather type
- Opens a reusable modal form for adding garments
- Opens a preview modal when a clothing card is clicked
- Supports closing modals by close button, overlay click, and Escape key

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- OpenWeather API

## Deployment

This project can be deployed with GitHub Pages.
